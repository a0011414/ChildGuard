## 毎週15分トーク準備支援アプリの名前案

**コンセプト**: チームのメンバーが**毎週の15分トーク**をするために、**スライド**と**話す内容**を作ることを支援するアプリ。開発者自身もプレゼンが得意ではなく学びたい想定なので、「自分もトークを組み立てながら学べる」製品にしていく。（あとで文字起こしを Cursor に読み込ませて手順・メソッドの構造化にもつなげる想定。）

### フォルダ名（コード・リポジトリ用。英数字が無難）

| 案 | 意味・イメージ |
|----|----------------|
| **TalkPrep** | Talk + Preparation。トークの準備。短くて分かりやすい。 |
| **WeeklyTalk** | 毎週のトーク。リズムが伝わる。 |
| **15MinTalk** | 15分トークであることが名前で分かる。 |
| **SlideTalk** | スライド＋トーク。作るものそのまま。 |
| **TalkCraft** | トークを組み立てる（Craft）。 |
| **Weekly15** | 毎週の15分。短い。 |
| **TalkBuilder** | トークを組み立てる人・ツール。**→ 採用**。 |
| **PrepTalk** | 準備してから話す。Prep + Talk。 |

### アプリ・製品名（表示用。日本語も可）

| 案 | 備考 |
|----|------|
| **TalkPrep** | フォルダと揃える場合。トーク準備のイメージ。 |
| **週間トーク** | 毎週のトーク。日本語でそのまま。 |
| **15分トーク準備** | 何の準備かはっきりする。 |
| **トーク準備ガイド** | 準備をガイドするアプリ感。 |
| **スライド＆トーク** | 作るもの（スライドと話す内容）をそのまま。 |
| **Weekly Talk** | 毎週トーク。英語表示用。 |
| **トークつくろう** | やわらかい。トークを作る支援。 |

**→ フォルダ・アプリ名とも TalkBuilder に決定済み。**
